MathShortsUploader

MathShortsUploader automatically produces original animated math explainer videos and publishes them to authorized social media accounts on a scheduled basis. It is operated by a single account holder (the developer) and connects only to that person's own TikTok, YouTube, and Instagram accounts.

What it does

The app uses mathematical animation software (Manim) and AI-assisted scripting to generate a short video explaining one math concept, with synthesized voiceover narration. Each video is rendered locally on the operator's own device. Before any video is published, the operator reviews it. For TikTok specifically, publishing is completed through a review screen that requires the operator to manually choose a privacy setting, manually choose who can comment, duet, or stitch the video, and confirm consent before the video is sent to TikTok.

How TikTok is used

The app uses TikTok Login Kit so the operator can authorize their own TikTok account via OAuth 2.0. Once authorized, the app uses the Content Posting API (Direct Post, file upload) to publish videos to that same account. No other TikTok account can be connected or posted to. Before each TikTok post, the app retrieves the account's current posting permissions from TikTok and presents them as the manual options described above.

Who uses it

MathShortsUploader is operated by its developer for their own educational video channel. It is not offered as a public multi-user service. Each connection requires the account owner to personally complete TikTok's OAuth authorization; the app never accesses any account that has not gone through that authorization.

Platform integrations

  • TikTok — Login Kit (OAuth 2.0) and Content Posting API (Direct Post / video.publish)
  • YouTube — YouTube Data API v3 (video upload)
  • Instagram — Instagram Graph API (video publish)
