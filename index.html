MathShortsAutomator is a personal automated content tool that creates short, animated math explainer videos and publishes them to a single, self-owned social media account on a fixed weekly schedule.

Each video is generated using mathematical animation software (Manim) and a synthesized voiceover, covering one math concept per video in a visual, intuitive style.

This tool publishes only to accounts that the operator owns and has personally authorized. It does not access or manage any other user's data, account, or content.

MathShortsAutomator — an independent educational content project.
